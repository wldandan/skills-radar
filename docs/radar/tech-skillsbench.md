## 核心技术

### 核心发现

SkillsBench 研究揭示了以下关键发现：

1. **人工精选的 Agent 技能能显著提升任务表现**：平均 +16.2% 成功率
2. **模型无法可靠生成对自身有效的技能**：自动生成的 Skills 几乎带来不了效果提升
3. **"少即是多"原则**：2-3 个技能效果最佳，过多技能反而降低性能
4. **Skills 系统平均拖垮 15% 任务成功率**：87% 任务至少在一个模型上完全没用

- 任务完成度
