## 一、技术概述

**Skill 召回**（也称 Skill 路由/Skill 调度）是智能体基于用户意图、上下文与场景特征，从 Skill 库中精准匹配最优 Skill/Skill 组合，完成从意图到执行的精准映射。

核心目标：实现"需求-场景-Skill"最优匹配，避免错配、漏召、过召，保障执行准确率与体验。

## 三、代表技术与项目

### 3.1 SkillRouter

破解大规模 Skills 选择难题的新范式。

**核心能力**：
- 多路召回 + 结果融合
- 基于强化学习的动态路由
- 支持千级规模 Skill 库

**技术亮点**：22.5% 性能提升，700 倍学习成本降低

### 3.2 SkillOrchestra

基于技能的 Agent 路由策略。

**核心能力**：
- 技能手册学习
- Pareto 验证筛选
- 模式感知 + 技能驱动的智能体选择

**技术亮点**：比 Router-R1 学习成本降低 700 倍，比 ToolOrchestra 降低 300 倍

### 3.3 Graph of Skills

千级规模 Skill 库的结构感知检索方案。

**核心能力**：
- 结构感知检索
- 43.6% 成功率提升
- 支持大规模 Skill 库

### 3.4 AgentSkillOS

生态级规模下技能的组织、编排与基准测试。

**核心能力**：
- 能力树构建
- DAG 执行计划编排
- 层级化检索
