## 一、技术概述

**Skill 评测**是贯穿 Skill 全生命周期的量化评估与质量定级体系，核心目标是构建可量化、可复现的多维度指标，以客观数据衡量 Skill 好不好用、稳不稳定、安不安全、有没有价值。

## 二、核心技术领域

### 2.1 多维评测

从多个维度评估 Skill 效果。

| 指标 | 描述 | 成熟度 |
|------|------|--------|
| 准确率 | 任务完成度 | 🟢 成熟 |
| Skill 召回率 | 正确选中目标 Skill 的比率 | 🟢 成熟 |
| Token 消耗 | 单次调用 Token 消耗 | 🟢 成熟 |
| 执行时延 | 任务完成时间 | 🟢 成熟 |
| CPSR | Cost-Per-Skill-Recall，每个召回的平均成本 | 🟡 成长期 |

## 三、代表技术与项目

### 3.1 skill-insight

openEuler 社区推出的多维评测与执行追溯平台。

**核心能力**：
- 覆盖效果、效率、成本多维指标
- 实时生成执行流程图
- 智能归因分析

**链接**：[AtomGit](https://atomgit.com/openeuler/witty-skill-insight)

**详细页面**：[skill-insight 详解](./tech-skillinsight)

### 3.2 SkillsBench

衡量智能体技能在多样化任务中表现的基准测试。

**核心发现**：
- 人工精选的 Agent 技能能显著提升任务表现（平均 +16.2%）
- 模型无法可靠生成对自身有效的技能
- 技能设计应遵循"少即是多"原则（2-3 个技能效果最佳）

**详细页面**：[SkillsBench 详解](./tech-skillsbench)

### 3.3 SkillProbe

用"魔法"打败"魔法"——用 Skill 审计 Skills。

**核心能力**：
- 安全漏洞检测
- 权限滥用检测
- 恶意行为识别

**详细页面**：[SkillProbe 详解](./tech-skillprobe)
