# Agent Skills 标准

> **分类**: Skill 管理 | **成熟度**: 🟡 成长期 | **综合评分**: 0.56

---

## 基本信息

**一句话描述**: 跨平台 Skill 格式与协议的开放标准 —— 解决"同一 Skill 在不同 Agent 框架上格式互通"问题。

**来源**:
- 社区开放标准
- 仅规范级别（非具体产品）

**链接**:
- 官网：[agentskills.io](https://agentskills.io/)

---

## 核心技术

### 标准化目标

定义跨 Agent 平台（Claude / Gemini / OpenAI / LangChain / Cursor 等）的统一 Skill 格式与协议，让一份 Skill 文件可被所有合规框架消费。

### 现状判断

- 已发布初版规范
- 部分工具（如 Skill Seekers）声称支持 20 个平台格式 —— 但**没有验证"转换后的 Skill 在目标平台上效果是否与源平台一致"**
- **格式兼容 ≠ 行为一致** —— 同一个 Skill 在 Claude Code 和 OpenCode 上即使格式兼容，执行行为也可能完全不同

---

## 技术细节

### 局限性

- 仅是**规范级别**，无强制约束力
- 不解决跨框架的**行为一致性**问题（只解决格式兼容）
- 缺乏权威治理机构和合规测试套件
