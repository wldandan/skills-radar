## 核心技术

### 事实分发标准

skills.sh 是 2026 年 Agent Skill 生态中**事实上的分发中心**：

| 维度 | 数据 |
|---|---|
| **索引 Skill 数** | 90,000+ |
| **支持的 Agent 框架** | 19 个 |
| **上线时间** | 2026-01-20 |

### 已发布官方 Skill 的公司（2026 Q1）

Vercel、**Prisma、Supabase、Stripe、Coinbase、Microsoft** 等。

> 当 Stripe 给自己的 API 发布"官方 Skill"时，相当于 npm 上的官方 SDK 包 —— 这是 Skill 生态从社区到工业的标志性转变。

### 与 SkillNet 的区别

- **SkillNet** —— 学术驱动 + Ontology 治理 + 5 维质量评分（偏研究）
- **skills.sh** —— 工业级分发 + 一键安装 + 跨 Agent 兼容（偏工程）

---

## 技术细节

### 主要能力

- 一键发现并安装 Skill
- 跨 19 个 Agent 框架格式适配
- 排行榜 / 趋势追踪

### 局限性

- **没有安全审计能力** —— Liu et al. 2026 发现 26.1% 社区 Skill 含漏洞，但 skills.sh 没有强制安全扫描
- 商业产品，未开源排行榜算法
- Vercel 主导，存在生态绑定风险
