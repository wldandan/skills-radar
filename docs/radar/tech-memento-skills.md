## 核心技术

### 核心原理

Memento-Skills 基于**Stateful Reflective Decision Process（有状态反思决策过程）**理论，核心突破是将**可执行技能**作为外部记忆的基本单元。

**Read-Write Reflective Learning 闭环**：
- **读取（Read）**：检索当前最相关的技能，用它来指导冻结的大语言模型
- **写入（Write）**：根据执行结果直接修改可复用的技能工件（代码、提示词、规范文件）

### 核心创新

- **技能路由器**：从"语义匹配"变成"任务成功"的优化目标
- **硬负例对比学习**：精准区分看起来很像但实际完全不同的技能
- **自动单元测试关卡**：确保技能修改不会越来越差

## 技术细节

### 主要能力

- 读写反思学习闭环
- 失败归因与针对性优化
- 自动生成全新技能
- 自动单元测试验证

### 实验结果

| 数据集 | 效果 |
|--------|------|
| GAIA 训练集 | 65.1% → 91.6%（三轮迭代） |
| GAIA 测试集 | 66.0%（+13.7个百分点） |
| HLE 训练集 | 30.8% → 54.5% |
| HLE 测试集 | 38.7%（是 baseline 的2倍多） |
| 技能库增长 | 5个 → 41个（GAIA）/ 235个（HLE） |

### 局限性

- 跨领域迁移能力有限
- 技能库收敛需要时间
